After years of harassing his colleagues with clean code and continuous integration, he was officially appointed “Code Cop” in 2006.

Despite the horrors of legacy systems, he is still enthusiastic about writing code. He is interested in code quality tools, code generation techniques and modern languages like Ruby or Scala. He helps teams of Software Developers to raise the quality of their work and improve their productivity. He regularly facilitates Coding Dojos and Code Retreats.
